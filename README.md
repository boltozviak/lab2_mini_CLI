# Мини обололчка с файловыми командами(Работает с пробивами + без тестов)
## МАИ - Программирование на Python - Лабоораторная №2
Реализация маленькой CLI с использованием библиотек Typer и Click_shell


## Функции
- `ls` - вывести содержимое каталога; `-l` - расширенный вид; `-a` - показать скрытые файлы
ls [path] [-l] [-a]

- `pwd` - вывести текущую директорию

- `cd` - переход между папками

- `cat` - вывести содержимое файла

- `cp` - скопировать файл; -r - рекурсивное копирование каталогов вместе с содержимым

- `mv` - перемещение или переименование файла/каталога

- `rm` - удаление файла; -r - рекурсивное удаление каталога со всем содержимым(требует подтверждения)
